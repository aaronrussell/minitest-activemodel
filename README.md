MiniTest matchers for `ActiveModel::Validations`.

## Support

This gem supports:

+ Ruby 1.9.
+ ActiveModel 3.2.x.

## Installation

Add this line to your application's Gemfile:

    gem 'minitest-activemodel', group: :test

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install minitest-activemodel
